# Under Bed Lighting Blueprint

A Home Assistant blueprint to automate under-bed lighting based on pressure sensors.

## Features
- **Triggers on Bed Exit**: Turns on lights when you get out of bed.
- **Night Mode Only**: Only runs when the house is in 'Night' mode.
- **Light Blocker**: Optional check to prevent triggering if another light is already on.
- **Auto-off**: Turns off when you return to bed or after a safety timer expires.
